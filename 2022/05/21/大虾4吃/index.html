炒大虾

发表于 2022-05-21 更新于 2022-10-04 分类于 热菜

介绍3种炒大虾的口味。油焖、蒜香、糖醋。最后附白灼大虾的简单吃法。

备料

香葱，鲜虾，其他无特殊

做法

  1. 新鲜大虾剪掉虾头备用，剪去虾尾巴
  2. 大虾去虾线（如果很新鲜，也可以不去虾线），开背
  3. 锅内下入宽油，放入虾头，中小火炸至油翻沫，表明虾头油已炒出，捞出控油
  4. 用稍高油温的虾油炸开背的大虾，炸至外科定型即可捞出备用。

接下来分别3种口味：

油焖

  • 料汁：胡椒粉，白糖，生抽，盐可选慎重，料酒
  • 煎好的大虾下锅加料酒，翻炒加姜葱，翻炒出香，加入料汁，小火焖5分钟，焖熟后汤汁浓稠即可出锅（多用小火）如果达到只见油不见汤汁为最佳，但是要小火控制好火候
  • 撒葱花上桌

注意：务必使用香葱而非大葱，葱要炒断生；这道菜成菜后无汤汁，只能看到一点油（但是油也不能放多），故曰油焖大虾。

蒜蓉 （最推荐，算是油焖大虾的蒜香版）

  • 大量大蒜切碎
  • 大虾加胡椒粉，葱姜炒香捞出备用
  • 锅内加一点虾油（不宜多），下入一半蒜蓉，虾油煎至表皮金黄，加盐，蚝油，生抽炒香，下入大虾翻炒，加白糖，加水烧开煮2分钟，放入其余的蒜蓉，翻炒均匀，加一点水淀粉勾芡（也可不加）
  • 撒葱花上桌

备注：这是金银蒜蓉的做法，和虾油绝配

糖醋

  • 料汁：白醋2，白糖，蚝油1，番茄汁3，淀粉1，盐，搅匀备用
  • 锅内下入大虾，炒熟（务必炒熟，前2种可以烧煮的时候二次成熟，这个不行）
  • 锅内酱汁煮浓稠，下入虾翻匀30秒，出锅
  • 点缀葱花上桌

注意，糖醋大虾汁水不宜太多，一定要收到汤汁浓稠，再下大虾；甜度不够可以再补白糖

备注

虾头剪去的时候可以往前剪一点，使得虾头油部分直接暴露。
大虾开背可以开深一点方便入味，保持不断即可。
炼虾油的时候油可以多放，多余的油炒饭、煎蛋、炒油菜都好吃。
本菜中所有炒制过程都是小炒，小料、大虾等轻轻炒出香味即可（大虾油炸可以大火快炸，壳肉略分离方便入味即可），火候一久发苦的话，大虾很难吃。

白灼大虾

鲜虾洗净
锅内加水，下入葱结姜块，煮开加一点料酒（料酒务必少放；如果虾极致新鲜，葱姜料酒这些都可以不加），多煮一会煮出葱姜味。
下入活虾，大火煮开，再煮30秒，此时虾肉口感最佳，鲜甜可口。
捞出后蘸料：香醋+盐+姜汁，可选生抽，日式酱油，芥末，芥末油，香葱，慎选香油，米醋、鱼露。但是东南有白醋+鱼露的吃法。
