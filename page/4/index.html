热干面

发表于 2022-09-16 更新于 2022-10-24 分类于 主食

备料

特殊食材：泡豆角，泡萝卜干

做法

  1. 碱面煮熟；

  2. 3勺芝麻花生混酱（如果太浓稠应提前用香油化开+2勺生抽老抽（混合）+适量辣椒油打底+蚝油（少量）+糖（少量），滴一点香醋，加入适量泡豆角和泡萝卜干，加面汤澥开

  3. 放入面条，拌匀即可，巨简单（可选自己喜欢的青菜）

碱面可购买梨花牌1kg热干面

备注

全用芝麻酱可能有苦味，最好用花生酱调和。
配煎蛋一起吃不好，发苦
比较咸，一般不额外加盐

tips

无

组装电脑记录

发表于 2022-06-23 更新于 2022-10-06 分类于 其他

组装电脑的准备工作：工具包一份，老虎钳/小扳手一个。老虎钳主要是机箱上有些铜柱需要根据主板孔位拆装，徒手拆不下来。


参考：

【装机教程】这可能是你能在网上找到最详细的装机教程_哔哩哔哩_bilibili

开机后，bios 打开 xmp 不然内存(1666速度以上的内存）速度不够

win10及以上系统安装程序要用Microsoft官网安装！或者msdn. 下载后PE引导（UEFI)。中间磁盘应当准备为未分配状态，可以用安装系统过程中删除功能（而非格式化功能）删除某一磁盘的分配状态，即可正常安装

Win7及之前的可以mbr分区后ghost安装

https://www.microsoft.com/zh-cn/software-download/windows10ISO/

https://msdn.itellyou.cn/

使用的PE系统是https://www.wepe.com.cn/。希望以后不会变味儿罢。

安装系统后，务必去购买主板及其他硬件的官网下载驱动！不要用公版驱动。特别留意主板、显卡。

注意：装好的电脑可暂时不联网，不登录，因为登录Microsoft账户虽然增加了安全性，但是在线Microsoft账户很麻烦，不如本地用户好用，登录之后还要手动转为本地用户。
