酸辣白菜

发表于 2021-11-01 更新于 2022-12-15 分类于 热菜

备料

大白菜
白醋

无特殊调料

做法

  1. 葱姜蒜切粒，蒜分2份
  2. 花椒粒爆香锅底后捞出，不然吃着碍口
  3. 干辣椒、葱姜蒜爆香锅底
  4. 下白菜稍微翻炒软（除调味外全程大火）
  5. 加入白醋，盐，将白菜炒8成熟
  6. 加生抽老抽、鸡精、蚝油调味，盐调口，最后加入香醋
  7. 大火琉璃芡，撒蒜末出锅

备注

加淀粉是勾芡是琉璃芡，不要太浓稠了。也不要太多水，因白菜会出水
这道菜不必放糖，白菜本身自带清甜，放糖则苦

毛血旺不正宗版-附鸡片鱼片焯水方法

发表于 2021-10-25 更新于 2025-02-04 分类于 热菜

备料

主料：毛肚、鸭血，各种时令蔬菜

配料：鸡胸肉，鱼片，肉片，火腿片等

调料：郫县豆瓣，火锅底料，干辣椒，芫荽

做法

  1. 毛肚切丝或切片；
  2. 蔬菜焯水煮熟放在碗底；
  3. 肉片类过水焯熟（见备注）铺在蔬菜上；
  4. 葱姜蒜花椒（干辣椒可选）爆香锅底；加入郫县豆瓣炒香，同时加入火锅底料炒化；
  5. 加入水（或者高汤），大火煮开，捞出料渣弃置；
  6. 盐、白胡椒粉调味；
  7. 下入鸭血煮熟；
  8. 下入毛肚，不超过15秒后关火；
  9. 倒入碗中，以水未没过全部食材为佳；
  10. 铺上生蒜、花椒面、花椒、辣椒面，油温6成，浇油；
  11. （如果有）锅内剩余汁水倒入碗内；
  12. 撒上芫荽，食用。

备注

  • 鸡片做法：鸡肉切片，用盐、味精、（一点点小苏打可选）一点料酒腌制10分钟左右，下沸水中焯熟即成。不要用酱油（如果是炒鸡或者炖鸡可以放酱油），不要勾芡；肉片同鸡片做法。
  • 鱼片做法：鱼肉请别人切成薄片，用盐、胡椒粉、一点料酒、葱、姜腌制10分钟左右，锅内烧水似开非开，鱼肉拍淀粉后逐一下入，煮至筷子夹着肉似断非断，捞出，口感最佳。
  • 严格说，正宗的毛血旺，最后的辣椒应当是刀口辣椒而不是辣椒面。

附上10月24日作品，还不错。毕竟是当年学厨时候学的第一个大菜。

tips

  • 不用担心油多或者花椒面多，这道菜就要大麻大辣大油。
  • 肉片都是提前滑熟，做菜时不用放料酒；

粉蒸肉

发表于 2021-10-18 更新于 2021-10-25 分类于 热菜

备料

主料：鸡肉片（或其他猪、牛、羊均可）

配料：土豆、南瓜、红薯均可

调料：蒸肉米粉，葱花

做法

  1. 肉类洗净切片，腌制；
  2. 配料切滚刀块，沥干水分；
  3. 蒸肉米粉与切好的肉片拌匀；
  4. 盘子控干水分，盘底铺上切好的块茎类食物，拌好的蒸肉在上面均匀铺一层。
  5. 上锅蒸60分钟（高压锅时间减半）；
  6. 取出撒上葱花，食用

备注

蒸肉米粉可根据自己口味，适量添加红腐乳、辣椒面、红油、花椒面、胡椒面等调味料

tips

肉类只能放一层，多了蒸好后不易分开

炖鱼汤

发表于 2021-10-08 分类于 热菜

备料

现杀河鱼（各种鱼类均可）

做法

鱼肉洗去血水，扣去腥鳃、牙齿，抽去腥线，刮鳞洗净粘液。

处理好的鱼肉加入姜、蒜、料酒、盐、白胡椒粉腌制

食用油润锅，撒入盐，升高温度后，下入鱼，鱼皮朝下煎熟；煎熟后加入葱姜蒜（可选花椒）炒香，加入水，炖鱼汤

想要奶白，就大火炖；想要清汤，就小火炖；火候越长越好。

用盐和胡椒粉调味。出锅前根据个人喜好，可选加入葱花、芫荽或香油。

备注

可加入豆腐、白菜、粉丝等材料，制成炖菜，同样鲜美可口。

tips

河鱼肉必须要新鲜的，才能鲜嫩不腥；如果鱼肉是现杀冻存的，只能保证不腥，但冻过后肉质变紧发硬，口感变差。
