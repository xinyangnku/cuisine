番茄虾滑粉丝煲

发表于 2024-09-25 更新于 2025-02-04 分类于 热菜

备料

香料：粉丝，番茄，虾滑，其余无特殊

做法

  1. 蒜切末，爆香锅底，炒香西红柿和番茄酱；粉丝泡发10分钟；葱花备好
  2. 调味生抽，盐，糖即可
  3. 勺子蘸凉水，下入买好的虾滑，咕嘟10分钟以上
  4. 下入泡软的粉丝略煮，撒入葱花即可

参考：抖音孩儿他姥姥-虾滑粉丝煲

备注

与一般番茄菜品有不同，本菜因为有虾滑，主要是蒜香
虾滑要煮10分钟以上

黑椒汉堡肉

发表于 2024-09-07 更新于 2025-02-04 分类于 热菜

备料

买来的汉堡肉未调味，油煎有诸多注意之处。
买黑椒酱

常用配料：藕，土豆，洋葱

做法

  1. 汉堡肉无需解冻，直接下小油锅
  2. 油温不要太高，正反面煎2分钟
  3. 发现开始缩小，立即关火捞出。
  4. 黑椒酱+黑胡椒碎+蚝油调味，下锅煮开，下入汉堡肉煮到汤汁浓稠即可。

备注

务必注意控制油温，快速出锅不要把汉堡肉煎皱缩。

麻辣小龙虾

发表于 2024-05-07 更新于 2025-02-04 分类于 热菜

备料

用冷冻的龙虾尾

备料火锅底料，干锅酱，小米辣，青麻椒，啤酒（实在不行可用白酒代替），一点香菜（不吃不放）

做法

  1. 龙虾尾解冻，小龙虾很脏，反复清水清洗几遍；焯水1分钟后捞出沥干备用；
  2. 葱姜蒜准备，干辣椒段，小米辣一半剁细，一半切圈后用。
  3. 锅内下入底油，下入葱姜蒜炒香，下入花椒；小火下入火锅底料炒出红油，下入干锅酱炒香；
  4. 下入焯好水的龙虾尾，炒匀；加入半罐啤酒，再加入没过龙虾尾的水，盖盖中火烧7-10分钟；
  5. 调味味精，蚝油，糖（糖可以多一点），生抽和盐（少放，各种酱够咸了），烧开收汁；
  6. 勾芡水淀粉，薄薄的一层芡汁即可
  7. 撒芫荽出锅。

备注

炸花生米

发表于 2024-03-07 更新于 2025-02-04 分类于 热菜

做法

  1. 先用温盐水泡10分钟，淘洗，去除灰尘/虫子
  2. 锅内烧油，冷油下入花生米，此时中小火炸，可以看到花生米冒小气泡，此时花生米正在丢失水分
  3. 待到花生米水分炸干，就开始出现炸裂的声音，此时开最小火
  4. 根据实际火力大小，再炸2-5分钟。此时一定要不停搅动，受热均匀。
  5. 捞出控油，随后放入大铁碗中颠簸快速冷却。冷却后即酥脆，此时皮颜色也变黑。

备注

千万不要超时，不要大火！

炖排骨_炖肉

发表于 2023-10-25 更新于 2025-02-04 分类于 热菜

备料

香料：葱姜料酒八角桂皮花椒

做法

红烧

  1. 锅内下入油，炒香料出香味，下入焯水过的猪肉煸炒
  2. 锅里水烧开后，放入排骨，加糖色、老抽，料酒（少放，务必蒸发干净），中大火烧10分钟上色
  3. 然后转入高压锅，设置50分钟（实际上汽35分钟）；或普通锅1小时45分钟即可。
  4. 转出后，转入锅中，调味烧制10分钟。

参考：https://v.douyin.com/iY121PWv/

红烧闻着就是香，清炖闻着就有一股猪肉本身的味道，虽然吃起来也没问题。
炖肉根据肉块大小，可设置25-35分钟

tips

  • 炖排骨，高压锅上气锁闭后1分钟，大概相当于普通锅3-4分钟
