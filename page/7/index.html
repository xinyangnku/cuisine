白菜炝锅面

发表于 2022-01-01 更新于 2022-02-03 分类于 主食

备料

大葱（多一些），生抽，胡椒粉，白菜，面条

做法

大葱切片炝锅

下入切好的白菜丝，翻炒变软（怕糊锅可以放一点水）

淋入一点水，烧开后淋入生抽炒熟

加入水，烧开后胡椒粉调味

两种加面条的方式：

  1. 沥出白菜，补水烧开，煮面
  2. 另起锅煮面，然后把半熟的面连汤带面倒入白菜中继续煮

煮开即可。

备注

炝熟的白菜和大葱的香甜，配合煮面条汤汁的浓郁，有妈妈的味道，那叫一个地道。

tips

推荐荞麦面条，不易煮烂

大葱多一点，炝锅味道浓

火腿肠煎蛋

发表于 2021-12-24 分类于 热菜

备料

无特殊食材

做法

  1. 火腿肠切片

  2. 锅内烧热，少量油润充分锅，下入火腿肠片先煎一下

  3. 将火腿肠片聚拢在锅底，然后打入荷包蛋

  4. 在蛋的上面再铺一层火腿肠片

  5. 最小火煎蛋一面定型后，用铲子翻过来煎另一面，出锅。

备注

本菜简单，所需材料少，咸鲜可口。

tips

如果用培根片，也可以先锅烧空气，下入培根片，这样就不用放油了。

全程最小火，防止煎糊

酸辣汤

发表于 2021-12-20 更新于 2022-02-03 分类于 汤

备料

大葱一根，香菜两根，紫菜少许，一小把虾皮，鸡蛋一个，老姜少许，干辣椒段少许，胡椒粉少许，陈醋少许，生抽少许。

做法

大葱，香菜各切成细丝，放在碗中备用，

切上少许的葱花和老姜片，

火腿一根切成细丝，放在碗中备用

先锅热，加入少许底油，先将葱姜下入锅内炒香

高温烹入陈醋大半勺，使其挥发出醋香味

加入清水四勺烧开

接下来咱们开始调味

加入三勺胡椒粉，两勺精盐，一勺鸡精调味，然后水开后，煮上二分钟

多次少量加入水淀粉勾芡

鸡蛋下入锅内，静置五秒钟后，再轻轻把鸡蛋搅开即可

葱丝，香菜，紫菜，虾皮火腿丝倒在锅内搅拌均匀

凉拌白菜帮

发表于 2021-12-14 更新于 2022-12-15 分类于 凉菜

备料

鲜嫩新鲜 大白菜
白醋
芫荽
干辣椒段

做法

  1. 白菜帮切细丝，冷水泡10分钟（吃的时候更脆爽，也可以不吃）；香菜洗净切段，辣椒洗净剪段
  2. 花椒粒爆香后捞出，花椒油倒辣椒段上激发出香味
  3. 盐：味精：白糖=1：1：4放入辣椒段中，加入足够量白醋调成料汁
  4. 白菜帮捞出控干水分，放入香菜段倒入料汁拌匀，5分钟后即可食用。

备注

糖盐比例根据个人口味增减
这道菜有可能腹泻，保持清洁
又名炝拌白菜丝

洋葱煎蛋

发表于 2021-12-05 更新于 2021-12-20 分类于 热菜

备料

无特殊食材

做法

  1. 洋葱切丁

  2. 按照一个洋葱配4个鸡蛋的比例，准备鸡蛋

  3. 锅内烧热，少量油润锅，下入洋葱炒熟，然后倒入蛋液，煎成饼翻面煎熟即可。

备注

本菜简单，营养丰富，打散的鸡蛋外酥里嫩。煎成饼状，可以撒孜然，可以刷酱，可以做蛋汤，可以吊高汤。可以配面条，可以卷煎饼/大饼。适配性极强。

tips

锅底撒少量盐，可以进一步减少粘锅的概率。

芹菜鸡丁

发表于 2021-11-29 更新于 2022-02-03 分类于 热菜

备料

芹菜、鸡脯肉、孜然粉、豆干（可选）

做法

鸡脯肉切丁，用盐、白胡椒、一点点料酒腌制；

芹菜洗净切丁备用；五香豆干（optional）切丁；

大量葱姜蒜切丁或片；

国内烧油，炸五香豆干，捞出控油；

下入油烧热，下入鸡丁滑熟；

控油，留下鸡丁，加入花椒、葱姜蒜炒香；

稍微加入一点水，烹入料酒、生抽，稍微烹制；

加入切好的豆干、芹菜、炒熟；

胡椒粉、孜然粉调味，根据咸淡加入盐，根据口味加入辣椒粉，关火，出锅。

备注

葱油拌面

发表于 2021-11-14 分类于 主食

备料

无特殊食材

做法

  1. 葱叶切丝，冷油下锅，小火炸干；

  2. 生抽：老抽 = 1：1，加入糖，酌情放入少量蚝油，加少许水，搅匀制成味汁；

  3. 葱油转最小火，倒入味汁，小火煮开至冒大泡即可关火；

  4. 煮面，捞出沥干，浇入葱油拌匀即可食用。
