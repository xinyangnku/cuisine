蚝油生菜

发表于 2023-01-09 分类于 凉菜

备料

无特殊食材

做法

  1. 蒜切末
  2. 糖，蚝油，酱油，淀粉，盐兑成料汁，注意不要太咸。
  3. 生菜切大长片后飞水捞出
  4. 热锅冷油下入蒜末小火炸香，下入料汁炒香，浇在生菜上即可

备注

咸甜适口，简单健康
